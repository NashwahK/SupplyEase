# SupplyEase

**SupplyEase** is a blockchain-based supply chain management portal built with **React + Vite** and powered by **Supabase**. It helps manage materials, products, and orders with a clean UI, order tracking, and real-time inventory insights.

## What It Does

- Dashboard with inventory KPIs and charts
- Order tracking with department-wise timelines
- Product & material management (with image uploads)
- RFID-based batch tracking simulation
- Search and filter tools for easier navigation

## Prerequisites

To run this project locally, you’ll need:

### Local Environment

- **Node.js v18+**
- **npm** (or yarn)

Check with:
node -v
npm -v

### Install Dependencies
npm install

### Create a .env file
At the root of the project, create a .env file to store the API keys and passwords

### Start the development environment
npm start


And you're all set! Contact us personally for log-in credentials and API keys.
